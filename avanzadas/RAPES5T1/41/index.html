Cuando la publicidad se hace con fines sociales o sobre temas de salud, educación o política se llama propaganda.
Cuando la publicidad se usa para promover algo o vender un producto, es decir, cuando tiene propósitos comerciales, se habla de anuncios publicitarios.
* Usan distintos tamaños y formas de letra para llamar la atención.
* Algunos anuncios incluyen un eslogan o una oración que sintetiza la idea principal para que los consumidores reconozcan y recuerden el producto o servicio.
* Incluyen textos breves que informan o describen los beneficios del producto o servicio que se anuncia.
* Presentan imágenes atractivas o impresionantes con el !n de crear el entorno del mensaje y atraer la atención de las personas.
